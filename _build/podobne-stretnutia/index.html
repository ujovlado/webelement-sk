Podobné stretnutia

Pár odkazov na stretnutia podobného druhu – naši kamoši.

V Bratislave:

GDG Slovakia

Skupina zameraná na propagáciu Google API a frameworkov, best practices a stretávanie ľudí, ktorí sa zaujímajú o Google technológie. meetup.com/gdg-slovakia

Java Group Slovakia

Skupina Java vývojárov na Slovensku. jug.sk

Rubyslava

Neformálne združenie priateľov elegantného programovania všeho druhu. rubyslava.sk

Slovak Scala Users' Group

Skupina pre každého, kto sa zaujíma o programovanie v Scale a súvisiace technológie. meetup.com/slovak-scala

A tiež pár v Brne:

Pyvo

Pravidelné stretnutie Python programátorov v Brne, Ostrave a Prahe. python.cz

BrnoJS

Stretnutia pre nadšencov JavaScriptu konajúce sa raz mesačne v Brne. brnojs.org

Brno PHP

Diskusie o nových veciach vo svete PHP, prípadne technológiách súvisiacich s web developmentom alebo PHP samotným. brnophp.cz
